# Beacon - Decentralized P2P Streaming

Beacon is a peer-to-peer streaming platform MVP that leverages user bandwidth to create a resilient, decentralized mesh network. By turning every viewer into a node, Beacon eliminates centralized server costs and democratizes streaming revenue.

## Key Features

*   **P2P Mesh Network:** Streamers and viewers relay content, creating a robust, distributed infrastructure.
*   **Incentivized Participation:** Earn credits by contributing bandwidth. Mandatory sharing ensures network health.
*   **Transparent Earnings:** Ad revenue and subscriptions are split directly among contributors based on their bandwidth contribution.
*   **Branding:** Distinctive Orange branding.
